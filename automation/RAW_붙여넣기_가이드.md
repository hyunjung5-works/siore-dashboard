# 재고 RAW 붙여넣기 가이드

매일 재고 RAW를 **탭에 통째로 붙여넣기만** 하면 대시보드에 자동 반영됩니다.
품목 매칭(아마존 ASIN, 국내 품목코드)과 Lot 합산은 **Apps Script가 자동으로** 처리합니다.

---

## 1회 준비 (최초 한 번)

✅ `RAW_국내재고`, `RAW_아마존재고`, `9.제품정보` 탭은 **이미 만들어져 있습니다.**

남은 것 하나: **Code.gs를 최신 버전으로 교체**하세요 (이 폴더의 `Code.gs`).
→ Apps Script 편집기에서 기존 내용 전체 삭제 → 붙여넣기 → 저장 → `runNow` 실행.

---

## 매일 하는 일 (30초)

### ① 국내재고
1. 아워박스에서 `WMS_재고현황` 파일 다운로드
2. 파일 열고 **전체 선택**(`Ctrl/Cmd + A`) → 복사(`Ctrl/Cmd + C`)
3. 시트 `RAW_국내재고` 탭 → **A1 클릭** → 붙여넣기(`Ctrl/Cmd + V`)

### ② 아마존재고
1. 아마존 셀러센트럴에서 FBA 재고 CSV 다운로드
2. 파일 열고 **전체 선택 → 복사**
3. 시트 `RAW_아마존재고` 탭 → **A1 클릭** → 붙여넣기

**끝.** 다음 자동 갱신(평일 09:50 / 15:00)에 반영됩니다.
지금 바로 보고 싶으면 Apps Script에서 `runNow` 실행.

---

## 붙여넣을 때 지켜야 할 것

- **헤더(첫 줄)까지 통째로** 붙여넣으세요. 스크립트가 헤더를 보고 컬럼을 찾습니다.
- **A1 셀**부터 붙여넣기 (다른 셀에서 시작하면 안 됨).
- 기존 내용 위에 그대로 덮어써도 됩니다. 행 수가 줄어도 문제 없습니다.
- CSV는 구글시트에서 **가져오기**보다 **엑셀로 열어 복사→붙여넣기**가 편합니다.

---

## 스크립트가 자동으로 하는 것 (검증 완료 — 21종)

**품목 매핑은 `9.제품정보` 탭을 읽습니다.** 하드코딩이 아니라서, 신제품이 생기면
**`9.제품정보` 탭에 한 줄(품목코드·ASIN·제품명·카테고리) 추가만** 하면 대시보드에 자동 반영됩니다.

**국내재고 (12종)** — `9.제품정보`의 국내 NMN·릴리프·RRP 품목코드로 매칭.
같은 코드가 여러 Lot로 쪼개져 있어도 **자동 합산** (예: 인텐시브 세럼 `1,879 + 2,880 = 4,759`)

**아마존재고 (9종)** — `9.제품정보`에 등록된 ASIN으로 매칭.
- FBA 가용재고 = `available` 열
- 예약중 = `Total Reserved Quantity` 열
- 최근 30일 판매 = `units-shipped-t30` 열

> `9.제품정보`가 매핑의 단일 기준입니다. 아마존 신규 ASIN이나 국내 신제품이 생기면
> 여기에만 추가하면 되고, `Code.gs`는 건드릴 필요 없습니다.

## 참고 — 국내 출고내역(통합현황)은 아직 자동화 안 됨

받은 통합현황(`통합_현황-....xlsx`)을 확인해보니, 안내 탭의 **유통처 8분류 규칙**(약국영업←공두균·한창훈 등)이 이 파일의 **주문자명과 맞지 않습니다.**
이 파일의 주문자는 최종 소비자(유동건·백지혜 등)라서, 발주 담당자 기준 8분류를 자동으로 못 만듭니다.

**확정된 분류 규칙 (다음 단계에서 적용):**
- 주문자명 **백지혜 → 본사 관리품** (별도 유통처로 분리)
- 나머지 8분류(약국영업·파라다이스·명동친한·B2B·마케팅·시딩)는 안내 탭 규칙 적용

**남은 확인 1가지 — 통합현황은 "하루치"입니다.**
받은 `통합_현황` 파일은 특정일 출고분이라, 매일 붙여넣으면 **그날 것만** 남습니다.
월 누계를 유지하려면 둘 중 하나가 필요합니다:
1. 통합현황을 **월 시작~오늘까지 누적**으로 뽑아 붙여넣기 (권장 — 단순)
2. 매일 붙여넣되 스크립트가 **누적 저장**하도록 설계 (복잡)

어느 방식으로 할지 알려주시면 출고도 재고처럼 붙여넣기 자동화하겠습니다.
그 전까지 출고내역은 기존 `8.출고내역` 탭을 그대로 사용합니다.
